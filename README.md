## 🕹️ Bevy - Snake

![Tela Principal](https://github.com/nfoj/bevy-snake/blob/main/img/bevy-snake.gif)


### 💾 Projeto
Esse pequeno projeto foi desenvolvido com o intuito de estudar a Game Engine Bevy que usa a Linguagem Rust, todo desenvolvimento foi desenvolvido de forma tranquila e rápida com base em outros projetos snake já desenvolvidos e publicados.


### ✒️ Sprites
O projeto não possui sprites, sendo usado apenas formas simples que aumentam proporcionalmente conforme o tamanho da tela, seguindo-se de forma responsiva.
